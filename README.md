
===================================MERN-PROJECT===================================

Deployed project link :- https://mern-first.web.app/

This project is a combination of React, Node, Express, MongoDB. Brief refreshers on the core technologies.

                              *** TO START PROJECT ***
        
        1. Go to the frontend folder.
        2. Run 'npm i react-scripts'.
        3. Run 'npm start'.
        
      
                            *** PROJECT DEPENDENCIES ***
For project to function following things must be installed

          node
          express
          app-env file
